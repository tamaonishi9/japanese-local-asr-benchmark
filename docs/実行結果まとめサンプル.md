# 実行結果サンプル

ローカル ASR モデルを同一音声で比較した結果。  
CPU は `moonshine/tiny-ja` を含む 4 profile、GPU は従来 3 profile を計測済み。

## 環境

- OS: Windows 11
- CPU: AMD Ryzen 7 9800X3D
- RAM: 32 GB
- GPU / VRAM: RTX 5070 / 12 GB
- テスト音声: 日本語読み上げ（約 30〜36 秒）
- 設定: language=ja、CPU=int8、CUDA=float16

---

## CPU 推論（device=cpu / compute_type=int8）

音声長: 35.9 s

- `moonshine_tiny_ja` — 推論 **0.520 s** / RTF **0.014** / ロード 1.2 s
- `small` — 推論 3.019 s / RTF 0.084 / ロード 1.2 s
- `kotoba_whisper_v2` — 推論 6.780 s / RTF 0.189 / ロード 1.8 s
- `large_v3` — 推論 13.440 s / RTF 0.375 / ロード 3.5 s

```
RTF 比較（CPU）  ← 短いほど高速、RTF=1.0 で音声長と同じ時間
moonshine_tiny_ja █                     0.014
small             ████                  0.084
kotoba_whisper_v2 ██████████            0.189
large_v3          ████████████████████  0.375
```

> RTF（Real-Time Factor）= 推論時間 ÷ 音声長。全モデル RTF < 1.0 → リアルタイム以下で完了。

---

## CUDA 推論（device=cuda / compute_type=float16）

音声長: 30.0 s

- `kotoba_whisper_v2` — 推論 **0.531 s** / RTF **0.018** / ロード 1.4 s
- `small` — 推論 0.733 s / RTF 0.024 / ロード 2.7 s
- `large_v3` — 推論 1.626 s / RTF 0.054 / ロード 2.7 s

```
RTF 比較（CUDA）
kotoba_whisper_v2 ███████               0.018
small             █████████             0.024
large_v3          ████████████████████  0.054
```

---

## CPU vs CUDA 速度比

- `small` — CPU 0.084 → CUDA 0.024（**3.4×**）
- `large_v3` — CPU 0.375 → CUDA 0.054（**6.9×**）
- `kotoba_whisper_v2` — CPU 0.189 → CUDA 0.018（**10.7×**）
- `moonshine_tiny_ja` — CPU RTF 0.014。CUDA 結果は未計測。

> CPU / CUDA は別録音の結果。速度比は参考値であり、厳密比較には同一保存音声での再計測が必要。

GPU があれば large-v3 でも **1.6 秒**で完了（十分実用域）。

## まとめ

- **速度優先（CPU）**: moonshine_tiny_ja が最速 RTF 0.014。large-v3 の約 25.9 倍速
- **速度優先（CUDA）**: 全モデル RTF < 0.06。kotoba が最速 RTF 0.018
- **精度優先**: large-v3 が句読点・英語表記ともに最安定
- **moonshine/tiny-ja**: CPU で既存 3 profile より高速。句読点と英語境界は弱いが本文末尾まで保持
- **kotoba-whisper**: GPU では最速だが CPU ではモデル名や表現の欠落あり
- **GPU 効果**: 既存比較対象では kotoba で 10.7 倍、large-v3 で 6.9 倍の高速化
