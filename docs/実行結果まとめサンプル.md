# 実行結果サンプル

ローカル ASR 3モデルを同一音声で比較した結果。  
CPU / GPU 両環境で計測済み。

## 環境

- OS: Windows 11
- CPU: AMD Ryzen 7 9800X3D
- RAM: 32 GB
- GPU / VRAM: RTX 5070 / 12 GB
- テスト音声: 日本語読み上げ（約 30〜34 秒）
- 設定: language=ja、CPU=int8、CUDA=float16

---

## CPU 推論（device=cpu / compute_type=int8）

音声長: 34.1 s

- `small` — 推論 **2.996 s** / RTF **0.088** / ロード 1.4 s
- `kotoba_whisper_v2` — 推論 8.029 s / RTF 0.235 / ロード 2.0 s
- `large_v3` — 推論 14.230 s / RTF 0.417 / ロード 3.6 s

```
RTF 比較（CPU）  ← 短いほど高速、RTF=1.0 で音声長と同じ時間
small             ████                  0.088
kotoba_whisper_v2 ███████████           0.235
large_v3          ████████████████████  0.417
```

> RTF（Real-Time Factor）= 推論時間 ÷ 音声長。全モデル RTF < 1.0 → リアルタイム以下で完了。

---

## CUDA 推論（device=cuda / compute_type=float16）

音声長: 30.0 s

- `kotoba_whisper_v2` — 推論 **0.531 s** / RTF **0.018** / ロード 1.4 s
- `small` — 推論 0.733 s / RTF 0.024 / ロード 2.7 s
- `large_v3` — 推論 1.626 s / RTF 0.054 / ロード 2.7 s

```
RTF 比較（CUDA）
kotoba_whisper_v2 ███████               0.018
small             █████████             0.024
large_v3          ████████████████████  0.054
```

---

## CPU vs CUDA 速度比

- `small` — CPU 0.088 → CUDA 0.024（**3.7×**）
- `large_v3` — CPU 0.417 → CUDA 0.054（**7.7×**）
- `kotoba_whisper_v2` — CPU 0.235 → CUDA 0.018（**13×**）

GPU があれば large-v3 でも **1.6 秒**で完了（十分実用域）。

## まとめ

- **速度優先（CPU）**: small が最速 RTF 0.088。large-v3 の約 4.7 倍速
- **速度優先（CUDA）**: 全モデル RTF < 0.06。kotoba が最速 RTF 0.018
- **精度優先**: large-v3 が句読点・英語表記ともに最安定
- **kotoba-whisper**: GPU では最速だが CPU では品質低下・切断リスクあり
- **GPU 効果**: kotoba で 13 倍、large-v3 でも 7.7 倍の高速化
